CartCheckoutCtrl
controller in module cart.module

Description

Contains checkout process. Customer can place an order by providing personal information, selecting a shipping method and a payment method. If selected payment method is registered as a module in the app, it initiates the online payment flow as the last step of the checkout procedure.
