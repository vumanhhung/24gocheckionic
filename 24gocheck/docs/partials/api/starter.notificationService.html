notificationService
service in module starter

Description

Initializes the notification service. This will register the device for push notifications. Subscribes the device for a scheduled notification with random greeting message notified in 86400000 seconds from now

Dependencies

  • $cordovaLocalNotification

Methods

  • init()

    Subscribes the device for a scheduled notification with random greeting message notified in 86400000 seconds from now

  • subscribeForPush()

    Initializes the notification service. This will register the device for push notifications.
