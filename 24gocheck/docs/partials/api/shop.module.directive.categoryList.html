categoryList
directive in module shop.module

Description

Widget to render category list.

Usage

as attribute
<ANY category-list
     mode="{string}"
     thumbnail="{boolean}">
   ...
</ANY>

Parameters

ParamTypeDetails
modestring

Template to render. Available options are simple or collapse. Default is simple

thumbnailboolean

Whether to show category image as a thumbnail or not. Default is false

Example

<category-list mode="simple|collapse" thumbnail="true|false"></category-list>
